About Woopsi - Objectives

Woopsi has four main objectives:

  • To reduce the amount of time taken to create homebrew GUIs
  • To improve the quality and complexity of homebrew GUIs
  • To provide a standard approach to homebrew GUI creation
  • To create a consistent look-and-feel across the homebrew applications that use the library
